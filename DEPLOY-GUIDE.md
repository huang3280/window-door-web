## 🚀 部署步骤（共5步）

### 第一步：注册 GitHub账号

1. 打开 https://github.com
2. 点击右上角 **Sign up**
3. 填写：
   - **Email**：你的邮箱（用于验证）
   - **Password**：密码（至少15位，或8位+数字+小写字母）
   - **Username**：用户名（英文/数字/-）
4. 验证邮箱（点击GitHub发的邮件链接）
5. 选择 **Free** 方案

---

### 第二步：创建仓库

1. 登录 GitHub
2. 点击右上角 **+** → **New repository**
3. 填写：
   - **Repository name**：`window-door-web`（或你喜欢的名字）
   - **Description**：门窗配单系统
   - **Public**（必须选Public才能用Pages）
4. 点击 **Create repository**

---

### 第三步：上传文件

1. 进入你创建的仓库
2. 点击 **Add file** → **Upload files**
3. **拖拽**这4个文件到上传区域：
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
4. 点击 **Commit changes**（提交）

---

### 第四步：启用 GitHub Pages

1. 进入仓库页面
2. 点击 **Settings**（设置）
3. 左侧菜单找到 **Pages**
4. **Source** 选择：`Deploy from a branch`
5. **Branch** 选择：`main` / `root`
6. 点击 **Save**

---

### 第五步：获取网址

等待1-2分钟，GitHub会生成网址：

```
https://你的用户名.github.io/window-door-web/
```

- 手机电脑都能用！
- 打开就能直接配单 ✨

## ⚠️ 常见问题

### Q: 网址打不开？
A: 等几分钟，GitHub需要部署时间。刷新页面再试。

### Q: 想修改内容？
A: 在GitHub仓库里点击文件，然后点击 ✏️ 编辑，修改后 Commit 即可。

### Q: 想改网址？
A: 网址由用户名和仓库名决定，改不了。但可以注册新账号或新建仓库。
