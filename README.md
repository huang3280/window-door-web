## ✨ 功能介绍

### 1️⃣ 首页
- 展示两个主要功能入口
- 点击卡片可快速进入对应功能

### 2️⃣ 帮我配单
- **选择类型**：窗 / 门
- **输入尺寸**：宽度、高度（单位：mm）
- **选择开启方式**：平开 / 推拉
- **点击"计算材料"**：自动计算框料、扇料、压线用量
- **点击"提交询价"**：复制询价信息到剪贴板，打开微信发送给销售人员

### 3️⃣ 来图报价
- **上传图片**：点击上传区域或拖拽图片文件
- **支持格式**：JPG、PNG
- **支持多图**：可一次上传多张图纸或料头照片
- **添加备注**：可选填补充说明
- **点击"提交报价请求"**：复制询价信息，打开微信发送图片和询价信息

## 📋 材料计算公式

- **框料** = (宽度 + 高度) × 2 ÷ 1000 （单位：米）
- **扇料（平开）** = (宽度 + 高度) × 2 ÷ 1000 （单位：米）
- **扇料（推拉）** = 宽度 × 2 ÷ 1000 （单位：米）
- **压线** = (宽度 + 高度) × 2 ÷ 1000 （单位：米）

## 💡 使用技巧

1. **手机上使用**：将文件夹传到手机，用手机浏览器打开 `index.html` 即可
2. **保存到桌面**：将 `index.html` 发送到桌面快捷方式，一键打开
3. **离线使用**：所有代码都在本地，无需联网就能用（除了联系销售需要联网打开微信）
4. **数据共享**：可以将整个 `window-door-web` 文件夹打包发给同事，他们也能用

## ⚠️ 注意事项

1. **联系销售功能**：点击"提交询价"会复制文字到剪贴板，然后需要您手动打开微信粘贴发送
2. **图片发送**：来图报价时，图片需要您在微信中手动发送（网页无法直接发送图片到微信）
3. **数据保存**：当前版本不会自动保存配置，每次刷新页面会清空（如需保存功能可后续添加）
